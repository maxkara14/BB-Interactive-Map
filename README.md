**🗺️ BB Interactive Map (Beta)**

Интерактивный радар и система пространственной памяти для SillyTavern. Расширение читает контекст последних сообщений и превращает их в наглядную 3x3 карту локации. Помогает и вам, и боту идеально ориентироваться в сцене.

**Что внутри?**

* 📡 **Сканирование окружения:** Нейросеть вытягивает из чата атмосферу, время суток и раскидывает персонажей с объектами по зонам.
* 📍 **Телеметрия зон:** Кликаете по сетке — и видите скрытые мысли персонажей, их настроение, отношение к вам и какие предметы валяются рядом.
* 🧠 **Память для ИИ:** Нажмите кнопку «Запомнить локацию» и расширение вошьет сводку карты прямо в контекст бота. Благодаря этому бот не забудет, что вы в тесном кабинете, а за спиной стоит охрана.
* 🚨 **Визуализация угрозы:** Сетка подсвечивается разными цветами и анимациями в зависимости от того, безопасно в зоне, напряженно или там полная бяда.
* 🎛️ **Удобное управление:** Весь функционал (запуск радара, просмотр того, что сейчас помнит бот, и очистка памяти) собран в компактном терминале.
* ⚡ **Кастомный API:** Возможность подключить отдельный API-ключ специально для генерации карты.
* 🛠 **Макросы:** Поддержка макроса {{bb_map}} для ручного внедрения сводки карты в ваши пресеты.

**⚙️ Установка:**

1. Откройте SillyTavern.
2. Перейдите во вкладку *Расширения -> Установить расширение*.
3. Вставьте ссылку на этот GitHub репозиторий: `https://github.com/maxkara14/BB-Interactive-Map`
4. Перезагрузите страницу (F5).
5. Кнопка запуска «Интерактивная карта» появится в меню «волшебной палочки» (в Расширениях).
